Readme Folder: .\examples
Readme Date:   2023-01-25

This folder contains example applications to Texture2Par to
both IWFM and MODFLOW models. Each folder contains a "run_t2p"
windows batch file that will call the Texture2Par executable
in the outside directory bin folder.

IWFM Models:
C2VSim - California Central Valley Groundwater-Surface Water Simulation Model 
SVSim  - Sacramento Valley Groundwater-Surface Water Simulation Model 

MODFLOW Model:
VirtualValley - Simple 1-layer MODFLOW 2005 synthetic model
